## 엣지케이스 & 리스크 (최소 3개)

**[R-1] JavaScript 비활성화 환경에서 핵심 콘텐츠 비노출**
- 모든 `.reveal` 요소의 초기 상태가 `opacity: 0`. JS가 비활성화되거나 IntersectionObserver 미지원 환경(구형 안드로이드)에서는 히어로 h1, CTA 버튼, 섹션 제목 등이 전혀 보이지 않음
- 대상 고객(봉덕동 인근 50~60대)이 구형 기기를 사용할 가능성 있음
- 권고: `noscript` 폴백 또는 JS 오류 시 `.reveal` 요소에 `.visible` 강제 추가 로직

**[R-2] 구글 폰트 CDN 로딩 실패 시 레이아웃 틀어짐**
- `Noto Sans KR`이 외부 CDN에서 로드됨. Wi-Fi 불안정한 매장 환경이나 구글 서버 지연 시 system-ui 폴백으로 표시
- 한글 시스템 폰트는 기기마다 달라 letter-spacing, 줄바꿈 위치가 달라져 레이아웃이 틀어질 수 있음
- 영업 시 클라이언트에게 직접 보여주는 상황이 매장 내부일 경우 첫인상 손상 가능

**[R-3] 클라이언트 검토 전 외부 공유 시 오정보 노출**
- 현재 시안에 클라이언트 확인 없이 작성된 영업시간·운영 경력·가상 리뷰 포함
- 전달 전 클라이언트(어머니) 검토 없이 링크가 외부에 공유될 경우 잘못된 정보 노출
- "수정 전 배포 불가" 조건을 명시적으로 전달 필요

**[R-4] 420~640px 구간에서 cat-grid 시각적 완성도 저하**
- 375px에서는 1열이지만, 420~640px 구간(소형 안드로이드, 구형 갤럭시)에서는 2열
- cat-img 고정 높이(156px)와 이모지 크기가 해당 구간에서 어색하게 표시될 수 있음
- 직접적 레이아웃 깨짐은 아니나 시각적 완성도 저하 가능
