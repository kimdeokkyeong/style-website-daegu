## 2. 타겟 업종 후보 (5개)

| 업종 | 홈페이지 필요성 | 평균 단가 | 접근 난이도 | 네이버 대체 가능성 |
|------|---------------|-----------|------------|-----------------|
| **네일샵/속눈썹샵/왁싱샵** | 높음 | 30~80만 원 | 낮음 (사장님 직접 운영) | 중간 (예약 기능 차별화 가능) |
| **헤어샵 (미용실)** | 높음 | 50~100만 원 | 낮음 | 중간 (브랜딩 차별화) |
| **피부관리실/메디스파** | 매우 높음 | 80~150만 원 | 중간 (의료 광고 주의) | 낮음 (시술 설명, 전후 사진 필요) |
| **PT샵/요가원/필라테스** | 높음 | 50~100만 원 | 낮음 | 중간 |
| **펜션/게스트하우스** | 매우 높음 | 80~150만 원 | 중간 (사진 퀄리티 중요) | 낮음 (플랫폼 수수료 탈출 욕구 강함) |

### 1순위 추천: 네일샵 / 속눈썹샵 / 왁싱샵

이유:
- 사장님이 1인 운영 많음 → 의사결정 빠름
- 예약·상담이 매출 직결 → 홈페이지 가치 설득 쉬움
- 인스타는 있지만 홈페이지는 없는 경우 많음
- 단가가 크지 않아 거절 리스크 낮음 (30~50만 원 첫 제안 가능)
- 노쇼 방지 예약 기능이 명확한 페인포인트

## 3. 업체 발굴 방법

### 구글맵 필터링 절차

```
1. 구글맵 → [지역명] + [업종] 검색 (예: "강남 네일샵")
2. 업체 클릭 → "웹사이트" 버튼 없음 = 타겟 후보
3. 리뷰 20개 이상 + 최근 3개월 이내 리뷰 있음 = 실제 영업 중
4. 영업시간 표시 정확함 = 사장님이 어느 정도 관리 중
5. 인스타그램 링크만 있고 홈페이지 없음 = 최적 타겟
```

### 타겟 제외 조건
- 리뷰 5개 미만 → 영업 불안정
- 최근 리뷰가 6개월 이상 없음 → 폐업 가능성
- 이미 홈페이지 있음 → 리뉴얼 영업 필요 (난이도 높음)
- 프랜차이즈 → 본사 통제로 독립 결정 불가

## 4. 아임웹 HTML 연동 방법 (확인됨)

- 디자인 모드 → 좌측 `+` → 코드 위젯 추가
- 위젯 우클릭 → 코드 설정 → HTML/CSS/JS 붙여넣기
- HTML 전체 100만자 한도 (실용상 충분)
- PHP/JSP/ASP 서버사이드 언어 불가 (클라이언트사이드만 가능)
- 아임웹 자체 jQuery/Bootstrap 이미 로드됨 → 중복 선언 불필요
- 전체 페이지 적용은 환경설정 → SEO → Body Code에 삽입

**결론**: Claude로 만든 HTML을 아임웹 코드 위젯에 그대로 붙여넣기 가능. 기술적 장벽 없음.

## 6. 만들면 안 되는 것 (최소 5개)

1. **복잡한 예약 시스템 (커스텀 개발)** — 아임웹 내장 예약 기능이나 카카오 채널 연결로 충분. 직접 구현하면 시간 대비 가치 없음.
2. **회원가입 / 로그인 시스템** — 로컬 업체에 불필요. 손님이 회원가입을 원하지 않음.
3. **관리자 CMS (복잡한 어드민)** — 사장님이 쓰지 않음. 아임웹 자체 관리자가 충분.
4. **다국어 지원** — 로컬 업체 타겟에서 과설계.
5. **복잡한 애니메이션 / 인터랙션** — 로딩 느려지고 모바일에서 깨짐. 클라이언트에게 설득 포인트 없음.
6. **블로그/콘텐츠 섹션 (초기)** — 첫 제안에 포함하면 복잡도 증가. 나중에 유지보수로 추가.
7. **쇼핑몰 기능** — 범위 이탈. 다른 비즈니스 모델.

## Research 단계 결론

**핵심 인사이트 3가지:**

1. **기술 장벽은 없다.** Claude → getdesign.md 와이어프레임 → 아임웹 코드 위젯 파이프라인이 확인됨. 하루 안에 제작 가능.

2. **가장 큰 리스크는 영업이다.** 사기 영업 전화가 많아 사장님들이 경계심이 높다. "먼저 결과물 보여주기" 전략이 이 리스크를 상쇄하는 핵심.

3. **포트폴리오가 없다는 게 현재 최대 약점.** 첫 제안에서 실제로 만든 시안이 없으면 설득 어려움. → Execute 단계에서 가상 업체 시안 1개를 먼저 만들어 레퍼런스로 활용.
